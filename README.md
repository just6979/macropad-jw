A single layout for the [Adafruit MacroPad RP2040](https://www.adafruit.com/product/5128).

* volume control on the knob, mute on press
* playback controls: rew, pause, ff
* undo/redo/paste-special
* cut/copy/paste
* fwd/back/calc

Based loosely on the code from https://learn.adafruit.com/macropad-hotkeys.
